关于 ANSI Common Lisp 中文翻译版

《ANSI Common Lisp》 —— 一本由 Paul Graham 编写的、一本非常棒的 Common Lisp 入门书籍，对于学习 Lisp / Functional Programming 、或者在《实用 Common Lisp 编程》或《On Lisp》之前阅读也非常有价值。

｜ 友情提示：《On Lisp》一书，知名 Lisper 田春已翻译完成。

本译文无正式获得授权，仅供同好学习交流之用，请勿用于商业用途。
